# 1. 운영체제란?
- 컴퓨터 HW를 관리하는 SW
- 프로그램이 실행되는 환경을 제공
- 컴퓨터 <-> 사용자 간 중간 역할

# 2. 운영체제가 실행되는 과정 (23~24p)
1. 컴퓨터 전원 켜기
2. CPU 레지스터는 모든 시스템 초기화
3. 부트스트랩(초기 프로그램) 실행
4. 부트스트랩(초기 프로그램)이 운영체제 커널 찾음
5. 운영체제 커널 메모리에 적재
6. 커널 실행
7. 시스템, 사용자 서비스 사용

# 3. 운영체제가 하는일
- 다중 프로그램 & 다중 태스킹
- 이중 모드 & 다중모드
- 자원관리
- 프로세스 관리
- 메모리 관리
- 파일 시스템 관리대용량 저장장치 관리
- 캐시 관리
- 입출력 시스템 관리
